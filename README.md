# 📝 Easy-Note-App

> A simple and efficient notepad application that gives you easy access to your notes.

[![Kotlin](https://img.shields.io/badge/Kotlin-1.9+-blue.svg?logo=kotlin)](https://kotlinlang.org/)
[![Android](https://img.shields.io/badge/Android-API%2021+-green.svg?logo=android)](https://developer.android.com/)
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Features

✨ **Quick Note Taking** - Create and save notes instantly  
📱 **Mobile Optimized** - Seamless experience on Android devices  
🚀 **Fast & Lightweight** - Minimal resource usage for maximum efficiency  
💾 **Easy Access** - Your notes are always just a tap away  
🎯 **Intuitive UI** - Simple and user-friendly interface  

## Prerequisites

- Android API 21 or higher
- Kotlin 1.9+
- Android Studio (for development)
